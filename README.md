# Wayland Switcher

Аналог Caramba Switcher для Wayland. Конвертирует последнее набранное слово (или выделенный текст) из одной раскладки в другую двойным нажатием Shift — и сразу переключает раскладку, чтобы продолжать писать на нужном языке.

```
Написал "ghbdtn" вместо "привет"?
Shift Shift → текст исправлен, раскладка переключена.
```

---

## Требования

| Пакет | Зачем |
|---|---|
| Python 3.8+ | интерпретатор |
| `python-evdev` | перехват клавиатуры |
| `wl-clipboard` | работа с буфером обмена (`wl-copy`, `wl-paste`) |
| `polkit` (`pkexec`) | запуск демона с правами root |
| `python-tkinter` | графический интерфейс |

---

## Установка

### 1. Установка зависимостей

**Arch Linux / Manjaro:**
```bash
sudo pacman -S python python-evdev wl-clipboard polkit tk
```

**Ubuntu / Debian:**
```bash
sudo apt install python3 python3-evdev wl-clipboard policykit-1 python3-tk
```

**Fedora:**
```bash
sudo dnf install python3 python3-evdev wl-clipboard polkit python3-tkinter
```

### 2. Скачать скрипт

```bash
curl -O https://your-host/wayswitcher-g2.py
# или просто скопируйте файл в удобную папку, например:
mkdir -p ~/.local/bin
cp wayswitcher-g2.py ~/.local/bin/wayswitcher.py
```

### 3. Сделать исполняемым (опционально)

```bash
chmod +x ~/.local/bin/wayswitcher.py
```

### 4. Автозапуск (опционально)

Чтобы GUI запускался при входе в систему, создайте `.desktop`-файл:

```bash
mkdir -p ~/.config/autostart
cat > ~/.config/autostart/wayswitcher.desktop << EOF
[Desktop Entry]
Type=Application
Name=Wayland Switcher
Exec=python3 /home/YOUR_USER/.local/bin/wayswitcher.py
Hidden=false
NoDisplay=false
X-GNOME-Autostart-enabled=true
EOF
```

> Замените `/home/YOUR_USER` на ваш реальный путь.

## Использование

### Запуск

```bash
python3 wayswitcher.py
```

Откроется небольшое окно управления.

### Интерфейс

```
┌─────────────────────────────────┐
│  Wayland Switcher           ☾   │
│                                 │
│  ● Остановлен      [Запустить]  │
│                                 │
│  Таймаут Shift: [0.4]  Сохранить│
│  Переключение раскладки: [▼]    │
│                                 │
│  Двойной Shift — конвертация    │
└─────────────────────────────────┘
```

- **Запустить / Остановить** — запускает фоновый демон. При первом запуске система спросит пароль (через `pkexec`) — это нужно для доступа к устройствам ввода.
- **Таймаут** — максимальный интервал между двумя нажатиями Shift в секундах. По умолчанию `0.4`. Если успеваете нажать два Shift быстрее — сработает конвертация.
- **Переключение раскладки** — шорткат, которым ваша система переключает язык. Выберите из списка или поставьте «Отключено».
- **☾ / ☀** — переключение тёмной/светлой темы.

### Как пользоваться

**Сценарий 1 — исправить последнее слово:**

Просто нажмите Shift дважды. Программа выделит последнее слово, конвертирует его и переключит раскладку.

**Сценарий 2 — исправить выделенный текст:**

Выделите нужный фрагмент мышью или клавишами, затем нажмите Shift дважды. Весь выделенный текст будет конвертирован.

**Направление конвертации** определяется автоматически по первому символу: если он русский — конвертируется в английский, и наоборот.

## Настройки шортката раскладки

Выберите тот вариант, который настроен в вашей системе:

| Вариант | Типичная система |
|---|---|
| Alt + Shift | Windows-стиль, часто на GNOME/KDE |
| Ctrl + Shift | Альтернативный вариант |
| Super + Space | GNOME по умолчанию (новые версии) |
| Отключено | Если переключение раскладки не нужно |

> Если ваш шорткат не в списке — его можно добавить в словарь `LAYOUT_SHORTCUTS` в начале скрипта.

---

## Конфигурационные файлы

Все настройки хранятся в `~/.config/wayland_switcher/`:

| Файл | Содержимое |
|---|---|
| `config.txt` | Значение таймаута (например, `0.4`) |
| `theme.txt` | Тема (`light` или `dark`) |
| `layout_shortcut.txt` | Название шортката (например, `Alt + Shift`) |

Можно редактировать вручную в любом текстовом редакторе.

## Как это работает

```
Двойной Shift
     │
     ▼
Демон (root) перехватывает нажатие через evdev
     │
     ▼
Сохраняет буфер обмена пользователя
     │
     ▼
Проверяет: есть ли выделение?
  ├─ Да → копирует выделенный текст
  └─ Нет → выделяет последнее слово (Ctrl+Shift+←) и копирует
     │
     ▼
Конвертирует текст (RU↔EN по первому символу)
     │
     ▼
Вставляет конвертированный текст (Ctrl+V)
     │
     ▼
Восстанавливает оригинальный буфер обмена
     │
     ▼
Симулирует шорткат переключения раскладки
```

Демон захватывает клавиатуру эксклюзивно (`grab`), поэтому все события сначала проходят через него и только потом пробрасываются в систему — никаких дублирующихся нажатий.

## Возможные проблемы

**Программа не видит клавиатуру**

Убедитесь что `pkexec` установлен и polkit-демон запущен:
```bash
systemctl status polkit
```

**Конвертация не срабатывает**

Попробуйте увеличить таймаут до `0.6`–`0.8` секунды — возможно, вы нажимаете Shift слишком медленно.

**Раскладка не переключается после конвертации**

Проверьте, какой шорткат настроен в вашей системе, и выберите соответствующий вариант в GUI. Для GNOME: Настройки → Клавиатура → Ввод.

**Окно GUI не открывается**

Убедитесь что установлен `python3-tk` (или `tk` на Arch). Проверьте:
```bash
python3 -c "import tkinter; print('OK')"
```

**`wl-paste` не найден**

```bash
# Arch:
sudo pacman -S wl-clipboard
# Ubuntu:
sudo apt install wl-clipboard
```
