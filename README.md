Wayswitcher (Caramba/Punto Analog for Linux)

Простая программа на Python для автоматической или полуавтоматической смены раскладки последнего напечатанного слова или выделенного текста в Wayland.

Особенности

Переключает раскладку по двойному нажатию Shift.

Работает в Wayland (проверено на GNOME и KDE).

Конвертирует выделенный текст или (если ничего не выделено) последнее слово.

Требования

Для работы требуются утилиты буфера обмена и библиотека для перехвата событий ввода:

wl-clipboard

python-evdev

Установка (Ubuntu/Debian)

sudo apt install wl-clipboard python3-evdev


Запуск

Программе требуются права root для доступа к /dev/input/ и сохранение переменных окружения пользователя для работы с буфером обмена Wayland.

sudo -E python3 wayland_switcher.py


Ограничения

В терминалах (Konsole, GNOME Terminal) автоматическое выделение слова (Ctrl+Shift+Left) может не работать. В этом случае выделите текст мышью перед нажатием Shift.
